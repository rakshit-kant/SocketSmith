About SocketSmith

SocketSmith is a Learning Project that I built from scratch to understand how HTTP Servers work under the hood. I will admit though that there was a fair amount of crashing out... Part of the Fun to be Honest.

I just wanted to learn how the VS Code Live Server Extension and Python Frameworks like FastAPI (Implemented in my Memory Notes Project) work under the hood.

Every Request is Accepted, Parsed, Routed, and Answered Manually using POSIX Sockets. (Atleast that's what Beej's Guide to Networking told me)
